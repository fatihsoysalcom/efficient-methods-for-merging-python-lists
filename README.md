# Efficient Methods for Merging Python Lists

This example demonstrates four different ways to merge lists in Python: the '+' operator, the 'extend()' method, list unpacking, and 'itertools.chain'. It also includes a performance benchmark comparing their execution speeds on larger datasets.

## Language

`python`

## How to Run

Run the script using Python 3:
python main.py
